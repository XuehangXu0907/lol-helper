# LOL Helper v2.0.0

## 🎮 功能概述

LOL Helper（英雄联盟助手）是一个功能强大的英雄联盟辅助工具，提供全自动化的游戏体验。

### 🆕 v2.0.0 新功能

#### 智能弹窗抑制系统
- **准备检查弹窗抑制**: 自动接受时隐藏准备检查弹窗
- **Ban阶段弹窗抑制**: 自动Ban时隐藏英雄选择弹窗
- **Pick阶段弹窗抑制**: 自动Pick时隐藏英雄选择弹窗
- **智能状态检测**: 根据游戏阶段自动判断是否需要抑制
- **错误恢复机制**: 连续失败后自动禁用并恢复，确保游戏正常进行

### 🚀 核心功能

#### 自动化功能
- ✅ **自动接受对局**: 检测到准备检查时自动接受
- ✅ **自动Ban英雄**: 在Ban阶段自动禁用预设英雄
- ✅ **自动Pick英雄**: 在Pick阶段自动选择预设英雄
- ✅ **弹窗抑制**: 自动化时隐藏游戏弹窗，提供无干扰体验

#### 系统集成
- ✅ **系统托盘支持**: 最小化到系统托盘，后台运行
- ✅ **开机自动启动**: 支持Windows开机自动启动
- ✅ **窗口管理**: 关闭窗口时可选择最小化到托盘
- ✅ **DPI适配**: 自动适配高分辨率显示器

#### 用户体验
- ✅ **中英文双语**: 支持中文和英文界面切换
- ✅ **智能布局**: 自适应窗口大小和文本宽度
- ✅ **状态监控**: 实时显示连接状态和功能状态
- ✅ **配置持久化**: 自动保存所有设置

## 🛠️ 技术架构

### 核心组件
- **LCU API集成**: 使用官方League Client Update API
- **PopupSuppressionManager**: 弹窗抑制核心引擎
- **SystemTrayManager**: 系统托盘管理
- **LanguageManager**: 多语言支持
- **AutoAcceptController**: 主控制器

### 安全特性
- 🔒 **官方API Only**: 仅使用官方LCU API，不修改游戏文件
- 🔒 **连接加密**: HTTPS安全连接到游戏客户端
- 🔒 **错误保护**: 多重保护机制防止功能异常
- 🔒 **自动恢复**: 检测到问题时自动恢复正常状态

## 📦 安装使用

### 系统要求
- Windows 10/11 (64位)
- 英雄联盟客户端
- 无需安装Java（安装包自带运行时）

### 安装步骤
1. 下载 `LOLHelper-2.0.0.msi` 安装包
2. 双击安装包，按照向导安装
3. 从桌面快捷方式或开始菜单启动程序
4. 启动英雄联盟客户端
5. 在LOL Helper中点击"连接"按钮

### 功能配置

#### 自动化设置
1. **自动接受**: 勾选"自动接受"复选框
2. **自动Ban**: 勾选"自动Ban"，点击"选择"按钮选择要Ban的英雄
3. **自动Pick**: 勾选"自动Pick"，点击"选择"按钮选择要Pick的英雄

#### 弹窗抑制设置
1. **准备检查**: 勾选"准备检查"以在自动接受时隐藏弹窗
2. **Ban阶段**: 勾选"Ban阶段"以在自动Ban时隐藏弹窗
3. **Pick阶段**: 勾选"Pick阶段"以在自动Pick时隐藏弹窗

#### 系统设置
1. **托盘**: 启用系统托盘功能
2. **关闭时最小化**: 关闭窗口时最小化到托盘而不是退出
3. **开机启动**: 开机自动启动LOL Helper

## 🐛 故障排除

### 常见问题

#### 连接失败
1. 确保英雄联盟客户端已启动
2. 检查客户端是否已完全加载
3. 尝试重启客户端和LOL Helper

#### 弹窗抑制不工作
1. 检查LCU API连接状态
2. 确认相应的自动化功能已启用
3. 查看状态显示是否有错误信息

#### 系统托盘图标不显示
1. 检查Windows通知区域设置
2. 点击"刷新托盘"按钮
3. 以管理员身份运行程序
