托盘图标说明
===========

为了使系统托盘功能正常工作，请在此目录下放置以下图标文件：

1. tray-icon.png - 16x16像素的PNG图标文件，用于系统托盘显示
2. icon.png - 应用程序主图标，推荐32x32或64x64像素

图标要求：
- 格式：PNG
- 透明背景
- 托盘图标：16x16像素（推荐）
- 主图标：32x32或更大

如果没有提供这些图标文件，程序将使用默认的蓝色圆形图标。

建议的图标设计：
- 简洁明了的设计
- 在小尺寸下清晰可见
- 符合应用程序的主题（LOL Helper）
- 可以使用英雄联盟相关的元素（如符文、装备等）
